# OlympusGCP-Data

High-intent cluster for GCP stateful storage services (Firestore, Bigtable, Datastore, Spanner).

## Domain: 61000 (Stateful Anchors)

This workspace provides:
1.  **Local Emulators**: Pre-configured Podman definitions for Google's official data emulators.
2.  **MCP Bridges**: Go-based servers that expose data manipulation tools to the fleet.
3.  **State Persistence**: Standardized volume management for local database state.
